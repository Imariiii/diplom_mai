\abstract % Структурный элемент: РЕФЕРАТ

\keywords{НАГРУЗОЧНОЕ ТЕСТИРОВАНИЕ, РЕЛЯЦИОННЫЕ БАЗЫ ДАННЫХ, СУБД, ПРОИЗВОДИТЕЛЬНОСТЬ, СРАВНИТЕЛЬНЫЙ АНАЛИЗ, ВЕБ-ИНТЕРФЕЙС, FASTAPI, NEXT.JS}

Объектом разработки является программный комплекс для сравнительного нагрузочного тестирования реляционных СУБД. Цель работы~--- создание инструмента, обеспечивающего автоматизированное тестирование, сбор метрик производительности и их анализ через единый веб-интерфейс.

Разработана многоуровневая клиент-серверная система, включающая серверную часть на основе FastAPI и клиентскую часть на основе Next.js с поддержкой MySQL, PostgreSQL и MariaDB. Реализованы генерация нагрузки по параметризованным сценариям, потоковая передача метрик в реальном времени, хранение истории прогонов, сравнительный анализ с применением статистических критериев, а также автоматическое резервное копирование и восстановление состояния баз данных с верификацией целостности. Развёртывание системы выполняется через Docker Compose.

Работоспособность подтверждена тестированием на двух наборах данных~--- Sakila и Brazilian E-Commerce~--- с тремя СУБД; всего проведено 12~прогонов с нулевой долей ошибок.

Результаты могут быть использованы при обоснованном выборе СУБД, оценке оптимизаций и предотвращении регрессии производительности.
