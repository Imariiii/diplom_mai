\abstract % Структурный элемент: РЕФЕРАТ

\keywords{НАГРУЗОЧНОЕ ТЕСТИРОВАНИЕ, РЕЛЯЦИОННЫЕ БАЗЫ ДАННЫХ, СУБД, ПРОИЗВОДИТЕЛЬНОСТЬ, СРАВНИТЕЛЬНЫЙ АНАЛИЗ, ВИЗУАЛИЗАЦИЯ ДАННЫХ, ВЕБ-ИНТЕРФЕЙС, FASTAPI, NEXT.JS}

Объектом разработки является программный комплекс для проведения нагрузочного тестирования реляционных СУБД и сравнительного анализа их производительности. Цель работы --- разработка системы для автоматизированного нагрузочного тестирования, сбора метрик производительности и визуального анализа результатов через единый веб-интерфейс.

В работе проведён аналитический обзор существующих инструментов нагрузочного тестирования, сформулированы требования к разрабатываемой системе. Спроектирована многоуровневая клиент-серверная архитектура. Реализованы: ядро генерации нагрузки с асинхронным выполнением SQL-запросов и виртуальными пользователями; механизм группы баз данных с автоматическим определением профиля схемы и генерацией пакетов запросов; модуль резервного копирования и восстановления состояния баз данных; подсистема визуализации метрик в реальном времени; модуль сравнительного анализа с применением критерия Манна--Уитни, поправки Бенджамини--Хохберга и размера эффекта Коэна~$d$. Работа системы подтверждена тестированием на двух наборах данных (Sakila и Brazilian E-Commerce) с тремя СУБД (MySQL, PostgreSQL, MariaDB) в режимах одиночного и серийного прогонов.

Результаты работы могут быть использованы инженерами по производительности, администраторами баз данных и разработчиками для обоснованного выбора СУБД, оценки эффективности оптимизаций и предотвращения регрессии производительности.
